A napari plugin for oocyte analysis.

----------------------------------

## Installation

You can install `napari-ooctyle-analysis` via [pip]:

    pip install napari-ooctyle-analysis

## Contributing

Contributions are welcome. Tests can be run with [pytest], please ensure
the coverage at least stays the same before you submit a pull request.
